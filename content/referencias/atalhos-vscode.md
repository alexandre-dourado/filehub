# Atalhos Essenciais — VS Code

## Edição

| Atalho | Ação |
|--------|------|
| `Ctrl+D` | Selecionar próxima ocorrência |
| `Ctrl+Shift+L` | Selecionar todas as ocorrências |
| `Alt+↑/↓` | Mover linha |
| `Ctrl+/` | Comentar linha |
| `Ctrl+Shift+K` | Deletar linha |

## Navegação

| Atalho | Ação |
|--------|------|
| `Ctrl+P` | Abrir arquivo rápido |
| `Ctrl+Shift+P` | Paleta de comandos |
| `Ctrl+G` | Ir para linha |
| `Ctrl+Tab` | Alternar entre abas |

## Terminal

```
Ctrl+` → abrir terminal integrado
Ctrl+Shift+5 → dividir terminal
```
