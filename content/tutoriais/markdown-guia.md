# Guia Rápido de Markdown

## Texto

**negrito** com `**texto**`  
*itálico* com `*texto*`  
~~tachado~~ com `~~texto~~`  
`código inline` com backticks

## Títulos

```
# H1 — título principal
## H2 — subtítulo
### H3 — seção
```

