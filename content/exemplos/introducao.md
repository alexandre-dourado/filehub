# Introdução ao Hub de Arquivos

Este é um arquivo de exemplo para demonstrar o **viewer de Markdown**.

## Funcionalidades

- Busca em tempo real
- Filtros por categoria
- Preview interno
- Favoritos com persistência

## Como usar

1. Adicione arquivos na pasta `/content`
2. Execute `node scripts/generate-index.js`
3. Faça o deploy no GitHub Pages

> Este hub é 100% estático e não precisa de backend!
